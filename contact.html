Contact

Network Packet Dictionary is a static reference website. If you want to report an issue, suggest a topic, or request a correction, please contact us using one of the methods below.

Email: rychee2connect@gmail.com

Suggested format

  • Page URL
  • What you observed (copy/paste Wireshark text is fine)
  • What you expected
  • Optional: environment (OS, tool version)

Privacy note

For privacy and security reasons, do not send packet captures that contain sensitive personal data. If you need to share an example, anonymize addresses and remove identifying payloads.

Response time

We read messages when available. Not all requests can be answered, but corrections and factual issues are prioritized.
